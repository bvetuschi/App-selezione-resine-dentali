# App Selezione Resine Dentali

**Strumento proprietario di supporto alla selezione e configurazione di resine per stampa 3D dentale.**

Applicazione web per la selezione guidata di materiali resinosi per stampanti 3D dentali, con confronto tecnico tra prodotti, compatibilità per tecnologia e indicazioni d'uso clinico e di laboratorio.
